The Stack

01

Frontend

Responsive layout using semantic HTML5, CSS Grid & Flexbox, mobile-first breakpoints, and vanilla JavaScript DOM manipulation.

  • HTML5 Semantic Landmarks
  • CSS Grid + Flexbox
  • Mobile-First Media Queries
  • Vanilla JS State Management
02

Backend API

Express.js RESTful API with GET, POST, PUT and DELETE endpoints, JSON responses, server-side validation, and proper HTTP status codes.

  • Express.js REST Endpoints
  • JSON Body Parsing
  • Input Validation Middleware
  • HTTP Status Codes (201, 400, 404, 500)
03

Database

SQLite database with a designed schema, relational integrity via primary keys, full CRUD mapped to HTTP methods, and parameterized queries against SQL injection.

  • SQLite Schema Design
  • Primary & Foreign Keys
  • CRUD → HTTP Method Mapping
  • Parameterized Queries (No SQLi)
